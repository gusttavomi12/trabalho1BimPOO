# 📱 Projeto SkyBlue - Aplicativo de Mensagens

## 📌 Descrição do Projeto

Este projeto foi desenvolvido com o objetivo de simular um aplicativo simples de envio de mensagens para a empresa **SkyBlue**, que está em crescimento e precisa melhorar a comunicação com seus clientes.

O foco do projeto é aplicar conceitos básicos de **Programação Orientada a Objetos (POO)** utilizando a linguagem Java.

---

## 🚀 Funcionalidade Implementada

* Exibir uma mensagem na tela simulando envio de mensagem para o cliente.

---

## 💻 Código Utilizado

```java
public class Main {
    public static void main(String[] args) {
        String mensagem = "Olá, cliente da SkyBlue!";
        System.out.println(mensagem);
    }
}
```

---

## 🧠 Explicação do Código

Neste código foi utilizada a linguagem **Java**, executada na IDE Eclipse.

* `public class Main`: define a classe principal do programa.
* `public static void main(String[] args)`: é o método principal onde o programa inicia.
* `String mensagem`: variável utilizada para armazenar o texto da mensagem.
* `System.out.println()`: comando responsável por exibir a mensagem na tela.

---

## 🛠️ Tecnologias Utilizadas

* Java
* IDE Eclipse
* GitHub

---

## 📌 Justificativa das Escolhas

Foi utilizado um código simples para demonstrar de forma clara o funcionamento básico de um sistema de mensagens.

A escolha da linguagem Java se deve ao fato de ser amplamente utilizada na Programação Orientada a Objetos, permitindo organização e fácil compreensão do código.

O uso da variável `mensagem` simula o envio de conteúdo ao cliente, enquanto o comando de impressão representa a exibição da mensagem no aplicativo.

---

## 📝 Conclusão

Este projeto, mesmo sendo simples, demonstra na prática como um sistema pode iniciar a partir de funcionalidades básicas. A partir dele, é possível evoluir para um aplicativo mais completo, com envio real de mensagens, cadastro de usuários e armazenamento de dados.

# 📌 ETAPA 2 – Diferenças entre IDEs e Editores de Código

As IDEs (Ambientes de Desenvolvimento Integrado) e editores de código são ferramentas utilizadas para facilitar o desenvolvimento de software, cada uma com suas características e níveis de recursos.

O **Eclipse** é uma IDE gratuita muito utilizada para desenvolvimento em Java. Ele oferece diversos recursos prontos, como compilação, execução e organização do projeto, porém pode ser menos intuitivo para iniciantes.

O **IntelliJ IDEA** é uma IDE mais moderna e inteligente, com sugestões automáticas de código e ferramentas avançadas que aumentam a produtividade. No entanto, sua versão completa é paga.

O **VS Code (Visual Studio Code)** é um editor de código leve e muito popular. Ele não é uma IDE completa por padrão, mas pode se tornar uma através da instalação de extensões. É muito utilizado por ser rápido, moderno e fácil de usar.

O **Notepad++** é um editor de código simples e leve, ideal para iniciantes ou edições rápidas. Ele não possui recursos avançados como compilação integrada ou gerenciamento de projetos, sendo mais limitado em comparação com as IDEs.

De forma geral, a principal diferença entre essas ferramentas está na quantidade de recursos: IDEs são mais completas e robustas, enquanto editores de código são mais leves e simples, exigindo configurações adicionais para funcionalidades avançadas.
